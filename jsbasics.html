JavaScript Basics

JavaScript is a cross-platform,object oriented scripting language.Javascript is extremly popular for variety of reasons.It is a small and lighweight language allowing maximum flexibility for developers to take in a bunch of different directions.JavaScript lives inside a host environmet(a web browser or Node server) ,it can be connected to the objects of these environments to provide programatic control over them.

  • Variable Declaration JavaScript variables are containers for storing data values-imagine a cup you fill with coffee,the cup holds the coffee,a variable holds a value.All JavaScript variables must be identified with unique names.These unique names are called identifiers.var x;
  • Variable Assignmet Assignment operators assign values to JavaScript variables-our cup now have coffee poured on it given our variable a value to hold.The = assignmet operator assign a value to variable.var x=10;
  • Data Types Data Types are important concept;to be able to operate on variables you need to know the data type. There are six data types which are JavaScript primitives:Boolean-True or False;Null -null aka nothing;Numbet-42 or 3.1459;String -"Coding Dojo Rocks!";Array -[1,'Coding',2,'Dojo'];and Object -{first_name:'Jane',last_name:'Doe'}
  • Conditionals when you write code,you want to perform different actions for different decissions-hitting differnt code blocks based on values and conditions that have been met.You can used conditional statements in your code to accomplish this .These are the following conditional statements:if a specified condition is true,do this code in our code block;else if to specify a new condition to test;if the firts condition is false;else we excute this block of code;
  • Loops There are many different kinds of loops in every programming language ,but all essentially do the same thing:they will repeat an action some number of times.Imagine you have to run a mile,well you run the track and then you stop.That is a loop!
  • FUnction functions are encapsulation of a code block.When we call our function this will run the code block.Think of it as a list of instructions.As an example imagine we are putting together a desk from IKEA, we open up the instruction manual and get started,first we screw the legs to the table top;next we place the table the right way up.Done! we finished our instructions. Sadly the are a ton more tables to do so lets cal our function over and over and over again.
